# Power BI_Healthcare_Provider_Dashboard_Project_1
Analyze the financial  performance of the healthcare center. Additionally, provide  insights on the healthcare providers.


# Requirements: 
1. KPIs and Trends: Identify and present the key performance indicators (KPIs) and significant trends in the dataset.    
2. Detailed Analysis: Ensure the analysis is thorough and detailed, covering all relevant aspects of the healthcare center's finance and the performance of healthcare providers.   
3. Dashboard Creation: Create a comprehensive dashboard consisting of 3 to 4 pages that includes:
   - Financial Overview: Summarize the financial health of the healthcare center.
   - Provider Insights: Analyze the performance and efficiency of healthcare providers.   

# Result:

* Designed and developed an **interactive Power BI analytics dashboard** to evaluate the financial performance of a multi-department healthcare center, integrating billing, medication, treatment, insurance, and room charge metrics into a unified decision-making dashboard.

Implemented **advanced data modeling**, **DAX measures**, and drill-down capabilities to reveal cost drivers across procedures, diagnoses, service types, departments, and patient demographics, enabling leadership to pinpoint high-revenue and high-cost segments.

Built predictive insight layers using historical trends (e.g., billing by procedure, insurance coverage impact, department-wise contributions), improving operational visibility and supporting data-driven resource planning for healthcare providers.

Developed user-centric interactive controls (filters for geography, year, race, insurance provider, etc.) to allow stakeholders to perform dynamic analysis and scenario exploration, enhancing the dashboard’s usability and business impact.
